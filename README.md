# 思维导图编辑器

## 网站简介
本项目是一款基于浏览器的轻量级思维导图编辑器，旨在提供流畅、直观的可视化思维整理体验。无需下载安装，打开网页即可开始创建、编辑与分享思维导图，助力灵感捕捉与逻辑梳理。

## 页面与功能
本项目共包含 **8** 个核心页面，涵盖完整的编辑与管理工作流：
1. **首页**：产品特性展示与快速启动入口。
2. **工作台**：核心画布，支持节点的增删改查、自由拖拽布局与样式调整。
3. **模板中心**：提供多场景预设模板，一键套用快速起航。
4. **文件管理**：本地历史导图列表，支持检索与批量操作。
5. **导出预览**：支持将导图导出为高清图片或通用文档格式。
6. **主题设置**：提供多种配色方案与界面风格切换。
7. **快捷键面板**：汇总全键盘操作指南，提升编辑效率。
8. **关于与帮助**：版本信息与操作指引。
